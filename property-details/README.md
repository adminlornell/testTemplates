# Property Details

This folder contains detailed markdown documentation for each property in the Lornell Real Estate portfolio.

## Files

Each property has its own markdown file named after its address (e.g., `1155-victory-hwy.md`).

## Contents

Each markdown file includes:

- **Property Overview** - Type, status, building class
- **Location** - Full address, market, submarket, coordinates
- **Building Details** - Size, year built, construction type, land area
- **Financial Information** - Pricing, rent, occupancy, taxes
- **Space Availability** - Available and vacant space
- **Building Features** - Parking, elevators, loading docks, utilities
- **Amenities** - Property-specific amenities
- **Contact Information** - Leasing and owner contacts

## Total Properties

20 properties documented across:
- Office properties
- Industrial properties
- Retail properties
- Flex space
- Multifamily properties

## Source Data

All information is extracted from `PropertyDetails.xml` (not included in this folder for security).

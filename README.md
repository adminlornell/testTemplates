# Lornell Real Estate Website

Professional commercial real estate website featuring 20 properties across Massachusetts and Rhode Island.

## Quick Start

### View Locally
Simply open `index.html` in your web browser to view the site locally.

### Deploy to Web

**Option 1: Netlify (Recommended - 5 minutes)**
1. Visit https://app.netlify.com/drop
2. Drag this entire folder to the browser
3. Your site is live!

**Option 2: GitHub Pages**
1. Create a GitHub repository
2. Upload all files
3. Enable Pages in repository settings
4. Site live at: `https://yourusername.github.io/repo-name`

## File Structure

- `index.html` - Main property listing page
- `marketing-plan.html` - Comprehensive marketing strategy
- `property-*.html` - Individual property pages (20 total)
- `PropertyDetails.xml` - Source data (exclude from public hosting)

## Features

- ✅ Responsive design (mobile-friendly)
- ✅ Property filtering by type, status, and city
- ✅ Individual property detail pages
- ✅ Professional marketing plan
- ✅ Contact information integration

## Customization

All pages use consistent styling defined in the `<style>` sections. To update:
- Colors: Modify CSS variables in `:root`
- Contact info: Update phone/email in footer sections
- Logo: Replace logo text or add image
